Welcome to the API

This API allows you to manage movies, series, and songs.

Endpoints

  • GET /movies - Retrieve all movies
  • POST /movies - Add a new movie (send JSON body)
  • GET /series - Retrieve all series
  • POST /series - Add a new series (send JSON body)
  • GET /songs - Retrieve all songs
  • POST /songs - Add a new song (send JSON body)
